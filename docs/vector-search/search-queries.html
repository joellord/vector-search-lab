Construct vector search queries

Just as you did in the previous step, you can explore the search results using an aggregation pipeline.

Querying with the aggregation builderâ€‹

  1. Navigate to the Collections tab of your database deployment.

  2. Navigate to the Aggregation tab from the navbar under your collection details.

  3. Click the Add Stage button and type $search in the select input.

  4. Add the following code for the $search stage.

    {
        index: "vectorsearch",
        knnBeta: {
            vector: `vector goes here`,
            path: "plot_embedding",
            k: 20
        }
    }

    The stage uses the vectorsearch index. Rather than using the 'text' operator as we did for full-text searches, we are using the knnBeta operator this time.

    For now, this stage will break. You will need to pass it a vector in the vector property.
